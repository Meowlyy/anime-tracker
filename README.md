# \# 🌸 Anime Tracker

# 

# Eine persönliche Anime-Watchlist-App – verfolge deinen Fortschritt, bekomme

# Benachrichtigungen für neue Episoden und importiere deine Liste direkt aus

# MyAnimeList, HiAnime oder AnimeKai.

# 

# 🔗 \*\*Live:\*\* \[anime-tracker.pages.dev](https://anime-tracker.pages.dev) <!-- Link anpassen, sobald Cloudflare live ist -->

# 

# \---

# 

# \## ✦ Features

# 

# \- 📺 Anime-Liste mit Status (geplant, läuft, abgeschlossen, pausiert, abgebrochen)

# \- 🔔 Benachrichtigungen bei neuen Episoden (via \[Jikan API](https://jikan.moe/))

# \- 📊 Statistiken zur eigenen Watchlist

# \- 📥 Import aus MyAnimeList (XML), HiAnime/AnimeKai (JSON/TXT)

# \- 💾 Export der eigenen Daten als JSON

# \- 🎨 Eigenes, minimalistisches Design

# 

# \## ✦ Tech-Stack

# 

# `HTML` · `CSS` · `JavaScript` (Vanilla, kein Framework)

# Daten werden lokal im Browser gespeichert (`localStorage`), externe Anime-Infos

# kommen live von der \[Jikan API](https://jikan.moe/).
